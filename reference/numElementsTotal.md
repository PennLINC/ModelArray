# Number of elements in ModelArray

Returns the number of elements in ModelArray, for a specific scalar

## Usage

``` r
numElementsTotal(modelarray, scalar_name = "FD")
```

## Arguments

- modelarray:

  ModelArray class

- scalar_name:

  A character, the scalar name (one of the existing scalar in
  `modelarray`)

## Value

number of elements in ModelArray, for this specific scalar
