# Number of input files in a ModelArray

Number of input files in a ModelArray

## Usage

``` r
nInputFiles(x, scalar = NULL)

# S4 method for class 'ModelArray'
nInputFiles(x, scalar = NULL)
```

## Arguments

- x:

  A ModelArray object

- scalar:

  Optional scalar name. Defaults to the first scalar.

## Value

Integer, number of input files (columns)
