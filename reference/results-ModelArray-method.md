# Access the results slot of a ModelArray object

Method for accessing the results slot of a ModelArray object.

## Usage

``` r
# S4 method for class 'ModelArray'
results(x, ...)

# S4 method for class 'ModelArray'
results(x, ...)
```

## Arguments

- x:

  An ModelArray object

- ...:

  Additional arguments. Currently accept analysis name (a character)

## Value

The results slot of the ModelArray object

Statistical results in this ModelArray object
