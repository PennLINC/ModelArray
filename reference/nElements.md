# Number of elements in a ModelArray

Number of elements in a ModelArray

## Usage

``` r
nElements(x, scalar = NULL)

# S4 method for class 'ModelArray'
nElements(x, scalar = NULL)
```

## Arguments

- x:

  A ModelArray object

- scalar:

  Optional scalar name. Defaults to the first scalar.

## Value

Integer, number of elements (rows)
